3.3 Variables

We can save the output of operations for later use by giving it a name using the assignment symbol <-. Read this symbol as â€˜getsâ€™, so x <- 5 reads as â€˜x gets 5â€™. These names are called variables, because the value they are associated with can change.

Letâ€™s give five a name, x then refer to the value 5 by itâ€™s name. We can then use the name in place of the value. In the jargon of computing we say we are assigning a value to a variable.

## [1] 5
 x * 2
## [1] 10
y <- 3
x * y
## [1] 15

This is of course of limited value with just numbers but is of great value when we have large datasets, as the whole thing can be referred to by the variable.

3.3.1 Using objects and functions

Functions are easy to spot, they are a name followed by a pair of brackets like mean() is the function for calculating a mean. The options (or arguments) for the function go inside the brackets:

## [1] 4
